## REST接口
----------------------------
+ /historytoday/{mon}/{day}                     历史上的今天
+ /feedupdate/all                               更新所有小说Feed
+ /feedupdate/single/{index}                    更新单个小说Feed
+ /ip/query/{ipQuery}/                          查询ip地址
+ /ip/open                                      打开ip功能
+ /ip/close                                     关闭ip功能(ip地址库更新时需要)
+ /ip/header                                    返回Header(json)
+ /novel/config ?cookieStatic=                  设置Cookie参数(已舍弃)
+ /novel/create ?url=                           新增小说
+ /novel/update ?name= &nid=                    更新小说(最小间隔6min)
+ /novel/closeBook ?nid=                        设为全本
+ /proxy/http ?proxyHost= &proxyPort=           设置http代理
+ /proxy/https ?proxyHost= &proxyPort=          设置https代理
+ /proxy/close                                  关闭代理
+ /proxy/test ?proxyHost= &proxyPort=           有效性测试
+ /proxy/testHide ?proxyHost= &proxyPort=       匿名性测试
+ /proxy/getProxyRest/{0,1,2} ?page=            获取免费代理(有效过滤后)
+ /proxy/getProxyAllRest/{0,1,2} ?page=         获取免费代理
+ /sitelink/spider/gentask ?link=https://m.nvshens.com/girl/22162/album/        建立下载任务
+ /sitelink/spider/run ?link=hhttps://m.nvshens.com/girl/22899/  (link可以为空) 开始下载任务
+ /sitelink/spider/query ?link=https://m.nvshens.com/girl/22162/album/      查询状态
:16888 下载图片访问
:8082  数据库访问(需要开防火墙)
:8090  webHook
