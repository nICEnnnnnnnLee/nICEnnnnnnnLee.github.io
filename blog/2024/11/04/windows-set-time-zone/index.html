Windows下通过脚本快速切换时区

2024-11-04
nIceLee

有时候，浏览器插件并不是很管用，我们需要切系统的时区。
这里提供一种使用脚本切换时区的方法。

关键命令行

  • 需要以管理员权限进行操作
  • 通过tzutil /g获取当前时区
  • 通过tzutil /l列出所有时区的值
  • 通过tzutil /s "China Standard Time"设置想要的时区
    例子里面对应的是(UTC+08:00) 北京，重庆，香港特别行政区，乌鲁木齐

样例脚本

脚本做了两件事：
1, 如果没有以管理员权限运行，那么申请权限
2, 如果时区为China Standard Time，那么改为Pacific Standard Time;
如果时区为Pacific Standard Time，那么改为China Standard Time

@echo off
if "%1"=="runas" (
    cd %~dp0
    for /f "delims=" %%a in ('tzutil /g') do (
        if /i "%%a"=="China Standard Time" (
            goto :changeTimezone1
        ) else (
            goto :changeTimezone2
        )
    )
) else (
    powershell Start -File "cmd '/K %~f0 runas'" -Verb RunAs
    goto :end
)

:changeTimezone1
echo "The timezone will be set to: Pacific Standard Time"
tzutil /s "Pacific Standard Time"
pause
goto :end

:changeTimezone2
echo "The timezone will be set to: China Standard Time"
tzutil /s "China Standard Time"
pause
goto :end

:end
